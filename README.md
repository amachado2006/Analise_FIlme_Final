#  Análise Exploratória: Indústria Cinematográfica (IMDb)

Este projeto apresenta um Dashboard interativo desenvolvido no **Power BI**, focado na análise de dados do IMDb. 

---

##  Objetivo do Projeto
Transformar dados brutos de mais de 30 mil filmes em insights visuais que ajudem a entender o que realmente move a indústria do cinema.

---

##  Perguntas de Negócio Respondidas
Para nortear o desenvolvimento dos visuais, o projeto focou em responder às seguintes questões:

1.  **Potencial de Lucro:** Quais são os gêneros cinematográficos que apresentam a maior receita bruta acumulada?
2.  **Qualidade vs. Prestígio:** Existe uma relação direta entre uma nota alta no IMDb e o número de vitórias no Oscar?
3.  **Investimento e Retorno:** Filmes com orçamentos astronômicos garantem mais indicações ao Oscar ou o sucesso crítico independe do custo?
4.  **Distribuição de Qualidade:** Qual a proporção de filmes considerados "Excelentes" (Nota > 8) em relação ao restante do catálogo?
5.  **Globalização:** Como está distribuído o acervo em termos de idiomas originais e diversidade?

---

##  Tecnologias Utilizadas
* **Power BI:** Tratamento de dados (Power Query), modelagem e visualização.
* **DAX:** Criação de medidas de performance e indicadores de status.
* **Dataset:** Dados históricos do IMDb (1960 - 2024).
